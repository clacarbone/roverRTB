roverRTB

a Return To Base library for remote controlled vehicles

STATUS

Currently the library is in alpha. Most of the functionality is there, but so is most of the debug and simulator code. Next step is polishing up the code, reducing the sources and separating debug and simulator code from the actual active code.

ASSUMPTIONS

A controlling C thread calling methods of this library in a pre-determined way. Anything having to do with timings and/or granularity is completely under control of the calling thread. No timers or other features are implemented by this library. Also it is assumed that the calling thread is able to provide the library with certain information:

  • position as a couple of FP values (either Cartesian or Lat/Lon)
  • heading as a single FP value (degrees true north)

Workings

The library provides for

  • storing sequential positions discarding already saved sequences (loop elimination)
  • backing up data to files (to be implemented)
  • heading to next target point until origin is reached
  • obstacle avoidance through laser scans
